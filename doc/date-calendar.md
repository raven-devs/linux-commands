# Date / calendar

```bash
# print current date and time
date "+%Y-%m-%dT%H:%M:%S %Z"

# utc time
date -u "+%Y-%m-%dT%H:%M:%SZ"

# display the calendar for the current month
cal

# display the calendar for the current entire year
cal -y

# display the calendar for a particular month and a year
cal $month $year

# display the calendar for a particular year
cal $year

# display a calendar for three consecutive months
cal -3
```
